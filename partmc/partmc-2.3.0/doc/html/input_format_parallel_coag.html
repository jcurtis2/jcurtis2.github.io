Input File Format: Parallel Coagulation Type

Read the specification for a parallel coagulation type from a spec file.

Parameters
[in,out]fileSpec file.
[out]parallel_coag_typeKernel type.

The output type is specified by the parameter:

  • parallel_coag (string): type of parallel coagulation — must be one of: local for only within-process coagulation or dist to have all processes perform coagulation globally, requesting particles from other processes as needed
