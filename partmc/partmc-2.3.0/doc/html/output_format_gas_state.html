Output File Format: Gas State

Write full state.

Parameters
[in]gas_stateGas state to write.
[in]ncidNetCDF file ID, in data mode.
[in]gas_dataGas data.

The gas state uses the gas_species NetCDF dimension as specified in the Output File Format: Gas Material Data section.

The gas state NetCDF variables are:

  • gas_mixing_ratio (unit ppb, dim gas_species): current mixing ratios of each gas species
