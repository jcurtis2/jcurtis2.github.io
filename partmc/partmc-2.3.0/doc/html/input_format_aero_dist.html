Input File Format: Aerosol Distribution

Read continuous aerosol distribution composed of several modes.

Parameters
[in,out]fileSpec file to read data from.
[in,out]aero_dataAero_data data.
[in,out]aero_distAerosol dist.

An aerosol distribution file consists of zero or more modes, each in the format described by Input File Format: Aerosol Distribution Mode
