Input File Format: Gas Material Data

Read gas data from a .spec file.

Parameters
[in,out]fileSpec file to read data from.
[in,out]gas_dataGas data.

A gas material data file must consist of one line per gas species, with each line having the species name. This specifies which species are to be recognized as gases. For example, a gas_data file could contain: 

 H2SO4
 HNO3
 HCl
 NH3
