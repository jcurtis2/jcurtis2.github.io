Input File Format:

Read the specification for a loss function type from a spec file and generate it.

Parameters
[in,out]fileSpec file.
[out]loss_function_typeFunction type. 
Loss Rate Function

The loss rate function is specified by the parameter:

  • loss_function (string): the type of loss function — must be one of: zero for no particle loss, constant for constant loss rate, volume for particle loss proportional to particle volume, or drydep for particle loss proportional to dry deposition velocity
